# Agentic AI Lab

A progressive four-day lab that builds from a simple rule-based agent all the way to a multi-step planning agent.

## Repository Structure
```
agentic-ai-lab/
├── day1/   Assignment 1 – Rule-Based Agent
├── day2/   Assignment 2 – Tool-Using Agent
├── day3/   Assignment 3 – LLM-Based Agent
├── day4/   Assignment 4 – Multi-Step Planning Agent
└── README.md
```

## Progression
| Day | Concept | Key Skill |
|-----|---------|-----------|
| 1 | Rule-based logic | Input → Intent → Action |
| 2 | Tool abstraction | Modular function calling |
| 3 | LLM routing | Prompt-based decision making |
| 4 | Planning pipeline | Task decomposition + sequential execution |

## Quick Start
```bash
# No external dependencies required for Day 1, 2, 4
python day1/agent.py
python day2/agent.py
python day4/agent.py

# Day 3 – optionally install Anthropic SDK
pip install anthropic
export ANTHROPIC_API_KEY=sk-ant-...
python day3/agent.py
```
